## Python GUI with Tkinter — Research Preparation

This repository documents my studies and practical experiments with **Tkinter**, the standard GUI library for Python. The goal is to develop the necessary skills to build graphical user interfaces for embedded systems, particularly touchscreen devices.

---

## Purpose

This project is part of my preparation for an international research program, where I will develop a **user touchscreen interface** for a prototype device.

---

## Exchange Program Information

* **Institution:** Technical University of Liberec
* **Faculty:** Faculty of Mechanical Engineering
* **Department:** Department of Manufacturing Systems and Automation
* **Period:** April 12 – June 16

---

## Study Topics Covered

* Tkinter fundamentals (widgets, layout, events)
* Responsive UI design for touchscreens
* Hardware interaction (GPIO, serial communication)
* Modular GUI architecture
* Real-time interface updates
